# ArrayClassLua
an optimized array class for lua with many methods

## Create a new array:

<Array>.new()


## Methods:

  - <Array>:push(value)
  - <Array>:pop()
  - <Array>:length()
  - <Array>:concat(otherArray)
  - <Array>:filter(fn)
  - <Array>:find(fn)
  - <Array>:forEach(fn)
  - <Array>:includes(value)
  - <Array>indexOf(value)
  - <Array>:map(fn)
  - <Array>:reduce(fn, initialValue)
  - <Array>:reverse()
  - <Array>:shift()
  - <Array>:slice(start, finish)
  - <Array>:sort(fn)
  - <Array>:join(seperator)
  - <Array>:keys()
  - <Array>:rotate(n)
  - <Array>:some(fn)
  - <Array>:unshift(value)
  - <Array>:values()
  - <Array>:fill(value, start, finish)
